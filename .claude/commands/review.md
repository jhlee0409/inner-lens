# 코드 리뷰

- [ ] undefined/null 처리
- [ ] 타입 단언(as) 최소화

### 에러 처리
- [ ] try/catch 적절히 사용
- [ ] 에러 메시지 명확성

### 성능
- [ ] 불필요한 리렌더링

- [ ] 단일 책임 원칙
